QuillQuest

Privacy Policy

Effective date: July 16, 2026

QuillQuest is a spelling & times-tables game for kids, built to be private by design. QuillQuest does not collect, store, or transmit any personal information.

The short version

  • No accounts. No sign-up, no login, no name or email required.
  • The game plays without a network. The full game — every level, both subjects, your default voice — works with no internet. Optionally, extra voices and recorded example-sentence audio download once from a content host and are then saved on the device; that request carries no personal information.
  • No ads. No tracking. No analytics. No ad networks, no third-party trackers, no cookies.
  • No personal data goes to anyone. We don't collect or share personal information. The optional audio download is an ordinary request for an audio file — the same file for everyone — that identifies no one.

What's stored on your device — and nowhere else

To run the game, QuillQuest saves the following locally, and never uploads it:

  • A player's age (used only to pick a starting level).
  • An optional first name and avatar you choose.
  • Game progress — levels, stars, XP, badges, streaks, words practiced.
  • Settings — theme, voice, sound, and reminder preferences.

Erase it anytime in Settings → Reset all progress, or by deleting the app.

The optional audio download

The core game is built into the app and needs no network. To keep the initial download small, some extra audio — additional voices, and the recorded "use it in a sentence" clips — is fetched once, on demand, from a content-delivery host (Cloudflare) and then cached on the device, so each clip downloads only a single time.

  • The request is a plain fetch of an audio file. It carries no name, no account, no advertising identifier, no identifier for your child, and nothing about which words, levels, or answers they practice.
  • If the device is offline, the game simply uses the device's built-in voice for that clip instead — nothing breaks.
  • The host serves the file over a standard, cookie-free request — no ad network, no analytics SDK, and no advertising or per-child identifier, just a request for the audio file.

Because these files are identical for every user and contain no personal data, this is content delivery, not data collection — nothing about your child is gathered, stored, or shared.

Notifications

The daily reminder is off by default and optional. If a parent turns it on, it schedules a single local notification on the device — no servers, no push tokens.

Children's privacy

QuillQuest is designed for children (grades 3–8, roughly ages 8–14). The app collects no personal information — and the only thing it ever sends is an anonymous request for an audio file that identifies no one — so there is no personal data to collect from a child. We do not knowingly collect information from anyone.
